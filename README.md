# Q: If I Want The Server App to Be Start When I Turn On Computer How I Can Do It?
**A:** If You Maked The Shortcut To Program You Can...

1. Press **Windows Key + R** And Type `shell:startup`
2. Move The Copy Of The Shortcut To Folder
3. It's Will Start When You Turn On Computer Now!
